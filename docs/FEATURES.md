# 잇닿 현행 기능 명세

## 서비스 경계

잇닿은 공고 및 지원 정보를 제공하고 병원의 채용·인력운영 업무를 지원합니다. 워커가 직접 공고를 선택해 지원하고 병원이 직접 검토·승인합니다. 임금은 병원이 워커에게 직접 지급하며 잇닿은 임금을 보관하거나 성공보수를 받지 않습니다.

## 워커 앱

- 현재 위치·등록 지역 기준 공고 탐색
- 날짜·시간·시급·부서 필터
- 공고 상세 확인 및 직접 지원
- 지원 상태와 병원 승인 결과 확인
- 병원 QR 기반 출퇴근
- 병원 등록 직원 초대 수락
- 휴가 신청과 잔여시간 확인

## 병원 관리자 앱

- 시프트 공고 등록·마감·재게시
- 지원자 목록 확인 및 병원 직접 승인
- 기존 직원과 공고 지원 인력 통합 관리
- 상시·기간제·임시·단기 계약 구분
- 직원 QR 출퇴근과 관리자 근태 보정
- 연차·반차·반반차·시간차 신청과 승인
- 월급·시급·일급 기준 세전 예상 급여
- 병원 등록 직원과 공고 지원 인력의 직접 지급 상태 관리
- 지급 검토 CSV
- 정액 SaaS 요금제와 별도 서비스 청구서

## 급여 자료의 범위

- 공고 지원 인력: 병원이 게시한 시급과 완료 근무기록 기준
- 병원 등록 직원: 월급·시급·일급과 완료 근태 기준
- 공통 상태: 검토 전 → 지급 승인 → 이체 준비 → 병원 지급 완료
- 승인된 금액과 근무시간은 스냅샷으로 보존
- 잇닿은 자동 공제·자동 송금·임금 보관을 하지 않음

법정수당, 중도 입퇴사 일할계산, 세금과 4대보험은 병원의 최종 노무·세무 확인이 필요합니다.
