- ✅ 확정 앵커 **11곳** (W여성병원 포함 — 기존 컨택 이력 최우선) / ⚠️ 심평원 확정 대기 **40곳**
- 확정 방법: 공공데이터포털 "심평원 전국 병의원 현황" CSV → Kurt 스크립트로 5개 구 필터·유형 분류·전화번호 추출 → v2 완성

## 컨택 전략
1. **1순위 요양병원 (구별 4~5곳, 총 ~25곳)** — 행정원장 직통, 인력공백 최심. atman 피치("빈 시프트 24시간 내 충원, 수수료 0")가 바로 꽂힘
2. **2순위 지역 종합·중소** — 동수원병원급. 외삼촌 파일럿 실적 후 원장 네트워크 소개로 진입
3. **대학병원(성빈센트·아주대) = 컨택 보류** — 레퍼런스 슬롯으로만 유지
4. 데모 시드 재생성은 v2 확정 후 (가상 조합명 → 실명 기반)
