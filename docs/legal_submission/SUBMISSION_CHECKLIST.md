# 국민신문고 제출 체크리스트

## 처리기관

- 고용노동부
- 경기지방고용노동청
- 고용관리과 이송 요청

## 민원 입력

- 제목: `병원 단기근무 구인정보 및 채용관리 SaaS의 직업정보제공사업 해당 여부 질의`
- 본문: `docs/LEGAL_INQUIRY_DRAFT.md`의 `민원 내용` 이하 붙여넣기
- 사업장 소재지: 경기도 수원시 권선구의 실제 사업장 주소 입력
- 답변 방식: 전자문서 또는 이메일

## 첨부파일

- `잇닿_직업정보제공사업_사전질의_첨부자료.docx`
- 사업자등록증 사본: 이미 발급된 경우에만 첨부

## 제출 후

- 국민신문고 접수번호 저장
- 접수 화면 PDF 또는 캡처 보관
- 답변 원문 PDF 보관
- 답변이 전화로만 오는 경우 서면 회신을 다시 요청
