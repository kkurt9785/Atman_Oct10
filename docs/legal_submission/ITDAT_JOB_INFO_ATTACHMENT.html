잇닿 서비스 설명자료

직업정보제공사업 해당 여부 사전질의 첨부자료

작성일: 2026년 7월 15일
사업장 예정 소재지: 경기도 수원시 권선구

잇닿은 무엇인가요?
병원이 단기근무 정보를 올리면 간호사·간호조무사가 내용을 보고 직접 지원하는 온라인 서비스입니다. 병원과 워커가 서로 선택하며, 근무가 끝난 뒤 임금은 병원이 워커에게 직접 지급합니다.
이 자료는 정식 서비스 시작 전, 관할기관에 실제 운영 예정 방식을 설명하기 위해 작성했습니다. 관할기관 답변에 따라 필요한 기능을 수정하고 신고 또는 등록 절차를 진행할 예정입니다.

1. 누가 서비스를 사용하나요?

이용자서비스에서 하는 일
병원필요한 근무 날짜, 시간, 장소, 업무, 임금 조건을 직접 작성하고 지원자를 직접 확인합니다.
워커여러 병원의 근무 정보를 직접 비교하고, 원하는 공고를 골라 지원합니다.
잇닿병원과 워커가 정보를 입력하고 확인할 수 있는 온라인 공간과 근태·지급관리 도구를 제공합니다.

2. 서비스 이용 순서

1단계　병원이 근무 정보를 직접 작성해 게시합니다.
2단계　워커가 공고를 검색하고 내용을 비교합니다.
3단계　워커가 원하는 공고에 직접 지원합니다.
4단계　병원이 지원자 정보를 보고 직접 수락하거나 거절합니다.
5단계　병원과 워커가 근로조건을 확인하고 직접 계약합니다.
6단계　잇닿에서 출퇴근 시간과 근무시간을 기록합니다.
7단계　병원이 워커 본인 명의 계좌로 임금을 직접 지급합니다.
핵심: 잇닿이 특정 워커를 골라 병원에 보내는 방식이 아닙니다. 워커는 공고를 직접 선택하고, 병원은 지원자를 직접 선택합니다.

아래 내용은 개발 중인 화면 구성을 문서용으로 쉽게 정리한 예시입니다.

잇닿은 지원자 순위를 정하지 않고, 특정 지원자를 채용하라고 병원에 권하지 않습니다.

5. 근로계약은 누가 하나요?

업무담당하는 사람
근무조건과 임금 제시병원
공고 선택과 지원워커
지원자 수락 또는 거절병원
근로조건 확인과 계약병원과 워커
화면·기록 기능 제공잇닿

잇닿은 근로계약의 당사자, 사용자 또는 대리인이 되지 않습니다. 병원과 워커 사이의 임금이나 근로조건을 대신 협상하지 않습니다.

6. 임금은 어떻게 지급하나요?

① 근무시간을 병원과 워커가 확인합니다.
② 잇닿은 예상 금액 계산과 지급 상태 기록을 돕습니다.
③ 병원이 자기 계좌에서 워커의 본인 명의 계좌로 직접 이체합니다.
④ 워커가 입금 여부를 확인합니다.
잇닿은 임금을 다루지 않습니다.
워커 임금을 받거나 보관하지 않고, 병원을 대신해 지급하지 않으며, 임금에서 플랫폼 수수료를 빼지 않습니다.

실제 개발 화면 2. 병원이 근무기록과 금액을 확인하고 직접 지급한 결과를 관리하는 화면입니다. 화면에도 잇닿이 임금을 보관하지 않는다는 점을 표시합니다.

7. 잇닿은 어떻게 수익을 얻나요?

SaaS란 사업자가 인터넷에서 업무용 소프트웨어를 이용하고 정기적으로 이용료를 내는 방식입니다.

과금 항목예정 방식
기본 이용료병원 또는 지점 단위의 월 정액 이용료
추가 사용량관리자 수, 활성 워커 수, 알림·API 등 시스템 사용량
워커 이용료받지 않음
채용 성공 수수료받지 않음
임금 연동 수수료받지 않음

병원이 내는 이용료는 특정 워커의 채용 성공 여부나 임금 액수와 연결되지 않는 소프트웨어 이용료로 운영할 예정입니다.

실제 개발 화면 3. 병원이 지점·관리자·활성 워커·공고·근태 사용량을 기준으로 SaaS 이용료를 확인하는 화면입니다. 워커 임금이나 채용 성공액과 분리된 요금임을 표시합니다.

8. 잇닿이 하지 않는 일

  • 특정 병원과 특정 워커를 골라 연결하거나 채용을 권유하는 일
  • 지원자를 평가해 순위를 정하거나 특정 지원자를 추천하는 일
  • 병원과 워커 사이에서 임금이나 근로조건을 조정하는 일
  • 근로계약 체결을 대신하거나 계약 당사자가 되는 일
  • 워커의 임금을 받거나 보관하거나 대신 지급하는 일
  • 워커에게 가입비·소개비·수수료를 받는 일

9. 관할기관 답변을 받은 뒤의 계획

  1. 직업정보제공사업에 해당하는 경우: 별지 제27호서식과 필요한 서류를 제출하고 신고확인증을 받은 뒤 운영합니다.
  2. 일부 기능을 바꿔야 하는 경우: 안내받은 기능과 운영정책을 수정한 뒤 다시 확인합니다.
  3. 유료직업소개사업에 해당하는 경우: 정식 운영 전에 필요한 인력·시설 요건을 갖추고 관할 시·군·구에 등록합니다.
이 자료를 제출하는 이유
잇닿의 실제 예정 운영방식을 숨김없이 설명하고, 정식 서비스 전에 직업정보제공사업 신고와 유료직업소개사업 등록 중 어떤 절차가 필요한지 정확한 안내를 받기 위함입니다.
