1. 3관법 통합 — 세 가지 시스템의 각도

자평명리 (子平命理)

  • 연해자평·자평진전·적천수·궁통보감
  • 십신·격국·용신·합충
  • "무엇이 일어나는가"
  • 구조와 원리

자미두수 (紫微斗數)

  • 12궁·14정성·4화·6길성·6흉성
  • 음력 + 시 기반
  • "어느 영역에서"
  • 인생 영역별 디테일

맹파명리 (盲派命理)

  • 빈주(賓主)·동정·실허·체용
  • 결과론적 사건 추적
  • "왜 그렇게 발현되는가"
  • 발현 메커니즘
세 관법을 한 번에 보는 이유 — 단일 관법은 60% 적중. 3관법 일치 사건은 75~80%. 본인 명조는 과거 8개 주요 사건이 3관법 모두에서 사후 설명 가능 = 매우 일관된 명조.

항목내용
명궁 (命宮)子宮 (수, 음) — 사색·내면·신중함·끈기. 토일주가 자궁 = "겉은 흙, 속은 물"
신궁 (身宮)未宮 (토, 음) — 일지·시지와 동일. 인생 후반 본 명조와 운명 일치 강화

을년 화록天機 — 지혜·총명·기술·연구·IT = 본인 재물 통로
을년 화권天梁 — 보수적 권위·자문·연장자 = 노년 권위
을년 화과紫微 — 제왕 명예·사회적 평판 = 임원·타이틀
을년 화기太陰 — 음·여성·어머니·딸 = 여성 가족 관계 미세 갈등 평생 관리
자미두수 핵심 메시지
재물은 천기(기술·IT) / 권위는 천량(자문) / 명예는 자미(임원) / 주의는 태음(여성가족 정서). 명리·자미두수가 같은 그림을 그림.

3. 맹파명리 빈주·실허 분석

구분위치해석
빈위(賓位)年·月 = 乙丑·癸未외부 환경·과거·조상·사회 / 칠살(외부 압박) + 편재(외부 돈)
주위(主位)日·時 = 己未·辛未본인·결과물·자녀 / 일간(나) + 식신(작품·기술)
천간 허(虛)乙·癸·己·辛표면 / 외부에서 보이는 모습 — 칠살·편재 머리 위
지지 실(實)丑·未·未·未본질 / 실제 무대 — 토 폭주, 미토 3중복

맹파의 결정적 진단 — 4가지

  1. 미토 3개의 무대 위에서 신금(작품·기술)을 들고 사는 사람 — 다중 정체성 운영자 (직장+사이드+개인작업)
  2. 계수 편재가 빈위 천간 = 돈은 외부에서 끌어오는 것 — 풀타임 단독 창업 비적합
  3. 을목 칠살이 년간 빈위 = 외부 권위가 평생 머리 위 — 무인대운 인목으로 정관화 → 외부 권력이 본인 권력으로 전환
  4. 축미충 = 빈주 충 = 외부와 본인 무대 마찰 — 정축대운 본격 발동
맹파 결정적 메시지 — 본인은 "외부 무대 위에 본인 작품을 얹어 외부 보상을 받는 명조". 외국계 IT + Atman + 자문직 형태가 명조에 정합.

13. 병화의 메커니즘 — 왜 좋은 일이 따라오는가

근거메커니즘
① 정인 = 도움·문서·명예기토 일간 + 병화 = 정인. 후원·자격·계약·신분 발현
② 천간 화 부재 보완원국 천간에 화 0개 → 병화 등장 = 외부 권위·후원 완성 (적천수 천간유결처득보)
③ 살인상생 (殺印相生)년간 을목 칠살 → 병화 → 기토. 위협이 도움으로 변환 (자평진전 길격)
④ 미월 조후 보완궁통보감 미월 기토 2차 조후 = 화. 조후득용
⑤ 병신합 — 식신 보호병화가 토를 따뜻이 만들어 신금이 묻히지 않고 드러남

과거 검증

시점병화 작용사건
2006 병술년병화 정인 + 술중 정화⭐ 대학 합격
2026 병오년병화 정인 + 자미 화과⭐ 재취업 정착, 정직 전환 예정
병화의 진짜 선물 — 통장 숫자가 아니라 신분·자격·계약·평판·신뢰로 들어옴.
2026 병오년은 반길반흉: 정인 안정·문서 확정은 강하나 인성과다로 식신 위협·자기표현 위축. 본업 안정 + 사이드 검증 균형이 정답.
